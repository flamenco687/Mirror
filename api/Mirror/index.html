Reflexâ€‹

</>
type Reflex = Configuration

Reflexes are containers that simply hold a value (using roblox attributes) in a studio-instance. The reflexes of a mirror represent that mirror's Mirror.Data in an instance tree so the client can easily access primitive values by making use of :GetAttribute(). If a reflex's value is nil it means that the real value is being held by the server because it cannot be set as an attribute value.
